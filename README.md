# Timo Nielsen Personal Website

A Next.js 14 project with Tailwind CSS and Framer Motion recreating the hero + scroll interaction inspired by **Topology.vc**.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000 to view the site.
